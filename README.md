# AOSF — Agentic Open Source Foundation Website

> The official website for the Agentic Open Source Foundation

## About

This repo hosts the AOSF website — a single-page application showcasing the foundation's mission, core functions, research, and community.

## Website

The website is built with:

- **HTML/CSS/JS** — Single `index.html` file, no build step required
- **Chart.js** — Interactive visualizations loaded from CDN
- **DM Sans + DM Serif Display** — Typography via Google Fonts
- **Light theme** — Professional foundation aesthetic with gradient accents and animations

### Sections

| Tab | Content |
|-----|---------|
| **Overview** | Mission, key capabilities, radar chart |
| **Core Functions** | 5 pillars: Repository, Rust Migration, Agentic Ops, Ethics, CELLO |
| **The Problem** | Gap analysis, foundation comparison table, vulnerability chart |
| **AI Code Provenance** | Provenance standard spec, JSON mockup, field explanations |
| **Get Involved** | Developer/Enterprise/Researcher paths, research links, CTA |

## Development

No build step required:

```bash
python3 -m http.server 8000
# Open http://localhost:8000
```
